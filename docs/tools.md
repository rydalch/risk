# Risk & Compliance Tools

Open-source and commercial tools for risk assessment, governance, and compliance.

## Risk Assessment & Quantification

### Binary Risk Assessment
- **Binary Risk Assessment (BRA)**: Structured questions for risk discussion: https://binary.protect.io/

### GRC Platforms
- **GigaChad GRC**: A comprehensive, modular, containerized Governance, Risk, and Compliance (GRC) platform built with modern technologies. Manage your entire security program from compliance tracking to risk management, third-party assessments, and external audits.: https://github.com/grcengineering/gigachad-grc
- **SimpleRisk**: Open Source GRC: https://www.simplerisk.com
- **Eramba**: Open Source GRC for risk and compliance: https://www.eramba.org
- **GigaChad GRC**: Open Source GRC platform: https://github.com/grcengineering/gigachad-grc
- **Claude GRC Plugin**: A plugin for Claude that assists with GRC tasks: https://github.com/mlunato47/claude-grc-plugin/tree/main
- **Free GRC using MS PowerApps**: https://store.securitybagel.io/

### Risk Visualization & Analysis
- **VCDB Explorer**: Visualization of vulnerability and compromise data: https://jpsturgis.shinyapps.io/vcdb_explorer/

## Vendor & Supply Chain Assessment

### Vendor Security Questionnaires
- **VSAQ**: Interactive questionnaire application (Google): https://github.com/google/vsaq

### Supply Chain Risk
- **HIC-SCRiM**: Health Industry Cybersecurity Supply Chain Risk Management Guide: https://healthsectorcouncil.org/hic-scrim/

## Assessment & Readiness

### CISA Tools
- **CSET (Cybersecurity Evaluation Tool)**: Ransomware readiness assessment: https://github.com/cisagov/cset/releases/tag/v10.3.0.0

## Data Analysis & Visualization

### Data Processing
- **Pandashells**: Bringing the Python data stack to the shell prompt: https://github.com/robdmc/pandashells
- **inetdata**: Download and normalize internet data from various sources: https://github.com/hdm/inetdata

### Data Visualization Libraries
- **dataviz book**: ggplot2 and visualization principles: https://github.com/clauswilke/dataviz
- **glasseye**: Present statistical analysis results in Markdown with D3 charts: https://github.com/coppeliaMLA/glasseye
- **ggcal**: Generate calendar plots from dates and fill values: https://github.com/jayjacobs/ggcal
- **Gradio**: Building modern dashboards with Python: https://towardsdatascience.com/building-a-modern-dashboard-with-python-and-gradio/

## AI-Assisted Security & Code Analysis

- **Trail of Bits Claude Code Skills**: A collection of security-focused skills for Claude Code, including smart contract security, entry point analysis, differential review, static analysis, and more: https://github.com/trailofbits/skills
